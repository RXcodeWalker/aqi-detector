AQI Detector

## What it does
This is a cheap, simple air quality detector that can be used to show how polluted the air is.

## Why this matters
There is a lot of pollution in the world and often times, especially in highly polluted areas like Mumbai, people do not realize how bad the air quality is because they are used to breathing it.

## Features
- CO₂ monitoring
- VOC sensing
- Visual alerts

## Hardware Used
- ESP32
- CO₂ sensor (and others)
- OLED display
